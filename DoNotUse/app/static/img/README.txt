Login page logos
================

Save the three logo images in THIS folder with these EXACT file names
(PNG with transparent or white background works best):

  1. gttc-logo.png            -> GTTC (Government Tool Room & Training Centre) logo
  2. samagra-shikshana.png    -> Samagra Shikshana Karnataka logo
  3. aim-niti.png             -> NITI Aayog · Atal Innovation Mission logo

They appear top-left on the login page, in this order:
  [ GTTC ] [ Samagra Shikshana ] [ NITI Aayog · AIM ]

Recommended height ~100-150px each (they are auto-scaled down on the page).
If a file is missing, that logo is simply hidden (no broken-image icon).
